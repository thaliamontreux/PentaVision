HTTPS required for installer

The installer can only be accessed over HTTPS.

If you are behind a reverse proxy or load balancer, ensure it terminates TLS and forwards HTTPS requests to this application.

For example, with an HTTP reverse proxy you should set the X-Forwarded-Proto header to https when the client connection is secure, so the installer can detect that the request is arriving over HTTPS.
